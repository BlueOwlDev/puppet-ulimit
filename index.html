Puppet Ulimit

Requirements

Tested on...

  • Debian 5 (Lenny)
  • Debian 6 (Squeeze)

Example usage

node /box/ {
  include ulimit

  ulimit::rule {
    'example1':
      ulimit_domain => '*',
      ulimit_type   => 'soft',
      ulimit_item   => 'nofile',
      ulimit_value  => '1024';

    'example2':
      ensure        => absent,
      ulimit_domain => '*',
      ulimit_type   => 'hard',
      ulimit_item   => 'nofile',
      ulimit_value  => '50000';
  }
}

Caveats

By default the module will purge any settings that are not managed by Puppet. While not advised you can disable this feature:

node /box/ {
  class { 'ulimit':
    purge => false,
  }
}
